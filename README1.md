# Mafia Dealer

## About

This is an app for playing mafia without cards for a group of friends

## Technology used:

- TypeScript
- React / NextJS
- Prettier
- Eslint
- Tailwind CSS
- i18n
- Service worker (PWA)

## Additionally:

- Mobile First
- Design (Figma)
- CI/CD
- Deploy (Vercel)
- Theme tailwind (Dark and Light)
